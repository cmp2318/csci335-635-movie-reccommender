# csci335-635-movie-reccommender
## Abstract  

This project develops a personalized movie recommendation system using the MovieLens 100K dataset. The objective is to predict user preferences and generate tailored movie suggestions based on historical rating data. We implement a collaborative filtering approach using matrix factorization to model relationships between users and movies, reccommending highly rated movies that are custom to a user's preferences. Model performance is evaluated using Root Mean Squared Error (RMSE).
